# 任务拆解模板

## 后端任务

```markdown
# [Task-201] 后端：实现登录接口

## 任务描述
实现 /api/login 接口，支持手机号+密码登录

## 技术实现路径
1. 接收 POST 请求，参数：phone, password
2. 查询数据库验证用户
3. 校验密码（bcrypt）
4. 生成 JWT Token（有效期 7 天）
5. 返回 Token 和用户信息

## 验收标准
- [ ] 接口响应格式正确
- [ ] 密码校验逻辑正确（bcrypt）
- [ ] Token 生成正确，包含 user_id 和 expire_time
- [ ] 错误处理完善（密码错误、用户不存在）
- [ ] 单元测试覆盖率 > 80%

## 预估工期
- 工作量：1 人天
```

## 前端任务

```markdown
# [Task-301] 前端：登录页面

## 任务描述
实现登录页面，支持手机号+密码登录

## 技术实现
1. 创建 Login 组件
2. 表单设计：手机号输入框、密码输入框、登录按钮
3. 表单校验：手机号格式、密码长度
4. 调用登录接口
5. Token 存储到 localStorage
6. 登录成功跳转到首页

## 验收标准
- [ ] UI 与原型图一致
- [ ] 表单校验正确
- [ ] 接口调用正确，Token 存储成功
- [ ] 错误提示友好
- [ ] 响应式布局（PC/Mobile）

## 预估工期
- 工作量：1 人天
```

## 测试任务

```markdown
# [Task-401] 测试：登录功能测试

## 任务描述
编写登录功能的测试用例并执行测试

## 测试用例
1. **正常登录**
   - 输入：正确的手机号和密码
   - 预期：登录成功，跳转到首页

2. **密码错误**
   - 输入：正确的手机号，错误的密码
   - 预期：显示"密码错误"提示

3. **账号锁定**
   - 输入：连续 5 次输入错误密码
   - 预期：账号锁定 30 分钟

## 验收标准
- [ ] 所有测试用例通过
- [ ] 接口自动化测试覆盖率 > 80%
- [ ] E2E 测试覆盖核心流程

## 预估工期
- 工作量：1 人天
```
