# TAPD 模板索引

## 可用模板

| 模板 | 用途 | 文件 |
|------|------|------|
| 需求分析 | 完整需求分析流程 | `requirement.md` |
| 用户故事 | Story创建模板 | `story.md` |
| 任务拆解 | 前端/后端/测试任务 | `task.md` |
| Bug报告 | Bug创建和分诊 | `bug.md` |
| 需求评审 | 评审检查清单 | `review.md` |
| 迭代规划 | Sprint规划模板 | `sprint.md` |
| 日报 | Daily standup | `daily.md` |

## 使用方式

根据需要读取对应的模板文件。

例如：
- 需要分析需求 → 读取 `guides/templates/requirement.md`
- 需要创建Bug → 读取 `guides/templates/bug.md`
