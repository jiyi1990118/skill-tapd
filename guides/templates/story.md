# 用户故事模板

```markdown
# 用户故事 - [Story ID]

## 故事描述
作为 [用户角色]
我希望 [功能描述]
以便 [业务价值]

## 验收标准
- [ ] 场景1：Given [前置], When [操作], Then [结果]
- [ ] 场景2：Given [前置], When [操作], Then [结果]
- [ ] 场景3：[异常场景]

## 技术实现
- **前端**：[技术栈] - [页面/组件]
- **后端**：[技术栈] - [接口/服务]
- **数据库**：[表/字段变更]

## 关联任务
- [Task-201] 前端：[任务名称]
- [Task-202] 后端：[任务名称]
- [Task-203] 测试：[任务名称]
```

## MCP 创建示例

```python
tapd_create_story(
  workspace_id="123456",
  name="作为老用户，我希望可以快速登录",
  description="""
## 故事描述
作为老用户，我希望可以快速登录，以便继续之前的操作

## 验收标准
### 功能验收
- Given 用户已注册且未登录
- When 用户输入正确的手机号和密码，点击"登录"
- Then 系统验证通过，跳转到首页，显示用户昵称

### 性能验收
- 登录接口响应时间 P95 < 2s

### 异常验收
- Given 用户输入错误密码 5 次
- When 用户第 5 次输入错误密码
- Then 系统锁定账号 30 分钟
  """,
  priority="high",
  owner="产品经理",
  iteration_id="迭代5"
)
```
